ans --> The purpose using comments in the HTML is
1. Improving code readability : it helps to the developer to understand the purpose of the specific section of the code, 
   making it easier for the developers to understand and maintain the code, especially when revisiting it at later date.

2. Temporarily disabling code : When we comment out the code, the commented code will not be rendered on the webpage/website.
   which helps to the developers for testing the multiple scenarios without permanently deleting code.

3. Adding Notes or Reminders : Comments can be used to leave the notes or the reminders for the other developers to
  understand the purpose of the provided section of the code.
