Ordered List of the HTML Tags
    1. address: address tag is used for providing the contact information of a person.

    2. aside:used to representing the sidebars of call-out boxes

    3. footer: used to representing the footer for the nearest content section

    4. header: used to representing the header for the content or the title for the content.

    5. h1 - h6: used to represent the hading having the different font-sizes on decreasing manner.

    6. main: used to represent the main content of a document or the page.

    7. main: used to represent the section of a page whose purpose is to provide the navigation links

    8. section: used to represent the section of a page or a document.

    9. search: used to represent the content of a document with forms, search or filtering functionality.

    10. div:It is a generic container for the content .

    11. hr:It represent the horizontal.

    12. li:Represents an item in a list. It must be contained in a parent element: an ordered list (ol), an unordered list (ul), or a menu (menu).

    13. menu:A semantic alternative to the ul but treated by the broweser as no difference that ul

    14. ol:Represents an order list of the items.

    15. p:Used to represent a paragraph.

    16. pre:Used to represent preformatted text. which is to be presented exactly as written in the HTML file.

    17. a:use to represent the hyperlinks in the application

    18. b:use to represent the important hading or to write the text in bold format

    19. br:used to represent the break between the text line. used for the block type elements to write start on the other line

    20. mark:used to write the text which highlighted or marked

    21. s:used to write the text with strikethrough

    22. code:used to write the code

    23. small: Represents side-comments and small print, like copyright and legal text, independent of its styled presentation.

    24. span: it is a inline container

    25. strong: Indicates that its contents have strong importance, seriousness, or urgency

    26. sub:Specifies inline text which should be displayed as subscript for solely typographical reasons.

    27. sup:Specifies inline text which is to be displayed as superscript for solely typographical reasons.

    28. img: used to represent the image

    29. audio: used to represent the audio

    30. embed: used to represent from the another webpage to the current webpage

    31. iframe: used to represent a nested browsing content. embedding the another HTML page into current one

    32. svg: svg is a element container that defines the new coordinate system and the viewport
